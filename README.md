# Toronto K-Market

A secondhand marketplace for the Korean community in Toronto.

## Overview

Toronto K-Market is a web application project inspired by local community marketplace platforms.  
The goal of this project is to create a user-friendly space where Koreans in Toronto can buy and sell used items, connect with local community members, and discover listings tailored to their area.

This project is currently in progress and will continue to evolve as features and product direction become more clearly defined.

## Goals

- Build a clean and modern marketplace UI
- Practice React and JavaScript fundamentals through a real project
- Create a localized experience for the Korean community in Toronto
- Explore features such as listings, search, chat, and user interactions
- Potentially expand with AI-powered features in future iterations

## Planned Features

- Home page with item listings
- Product detail page
- Category browsing
- Search and filtering
- User profile page
- Posting new listings
- Favorites / saved items
- Chat interface for buyers and sellers
- Location-based community marketplace features
- AI-assisted features to be decided
